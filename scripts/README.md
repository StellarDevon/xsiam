# scripts/

One-off development and utility scripts. Not part of the build.

| File | Purpose |
|------|---------|
| `screenshot.cjs` / `screenshot.mjs` | Playwright screenshot capture for UI review |
| `shot_sidebar.cjs` | Sidebar-specific screenshot helper |
| `web_screenshot.cjs` | Screenshot helper used from web/ context |
| `check_css.py` | One-off CSS audit script |
| `fix_i18n2.py` | i18n string repair (run once) |
| `fix_i18n_damage.py` | i18n rollback helper |
| `i18n_zh.py` | Chinese i18n string extraction |
